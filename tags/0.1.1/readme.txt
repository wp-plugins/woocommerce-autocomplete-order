This plugin allows to automatically set any order for any virtual product on Completed status right after a successfull payment (e.g. with PayPal).

== Description ==

WooCommerce will mark virtual-downloadable orders as "completed" after successful payment by design. If you want to extend this functionality too virtual only product, this is the plugin for you!

== Installation ==

1. Upload the plugin's folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Flag as "Virtual" the product you want to be unlocked for users upon successfull payment without any further (manual) action
